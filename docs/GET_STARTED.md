# Get Started (CLEAN)

1) **Website (Next.js + TS)**
```bash
cd apps/website
npm install
npm run dev
```
→ öffnet http://localhost:3000

2) **Studio (Vite + JS)**
```bash
cd apps/studio
npm install
npm run dev
```
→ öffnet http://localhost:5173

3) Optional: Shared Packages bauen (später; aktuell nur Platzhalter)
